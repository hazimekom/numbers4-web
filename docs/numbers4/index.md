# ナンバーズ4とは？基本ルールと当選の仕組み

このページでは、数字選択式宝くじ **ナンバーズ4** について、基本的な仕組み・当選方法・制度上の特徴を整理して解説します。
初めてナンバーズ4を知る方が制度を理解するための参考情報としてご利用ください。

※本ページは当選を保証するものではありません。

## ナンバーズ4の基本ルール

ナンバーズ4は、購入者が **4桁の数字** を選択する宝くじです。
抽選で決定された4桁の数字と一致した場合に当選となります。

### 基本情報

- 選ぶ数字：0000〜9999 の4桁
- 価格：1口 200円
- 抽選日：原則として平日（月〜金）

抽選は各回独立して行われ、過去の結果が次回の結果に影響することはありません。

## 当選方法の種類（買い方）

ナンバーズ4には、主に次の3つの当選方法があります。
当選条件の違いにより、当選確率や当選金額が異なります。

### ストレート

数字と並び順が完全に一致した場合に当選となります。

- 特徴：当選条件が最も厳しい
- 備考：当選金額は比較的高くなりやすい

### ボックス

数字が一致していれば並び順は問わない当選方法です。

- 特徴：ストレートより当選しやすい
- 注意：
	- 数字の組み合わせ（重複の有無）により当選パターン数と当選金額は変動します
	- 「1111」のような4桁すべて同一数字の場合は並び替えが存在しないため、ボックス購入はできません

### セット

ストレートとボックスを同時に対象とする当選方法です。

- 数字と並び順が一致 → セットストレート当選
- 数字のみ一致 → セットボックス当選
- 特徴：両方の当選条件を同時に確認できる方式

## 当選方法の比較（参考）

| 当選方法 | 当選条件 | 特徴 |
|---|---|---|
| ストレート | 数字・並び順が完全一致 | 当選条件が最も厳しい |
| ボックス | 数字が一致（順不同） | 比較的当選しやすい |
| セット | ストレート＋ボックス | 条件を同時に確認 |

※当選確率・当選金額は回ごとの販売状況により変動します。

## 数字の組み合わせと出現パターン

過去の当選番号を **数字の重複パターン** で分類すると、次のような傾向が見られます。

| パターン | 例 | 過去データでの出現傾向 |
|---|---|---|
| シングル | 1234 | 比較的多い |
| ダブル | 1123 | 比較的多い |
| トリプル | 1112 | 少なめ |
| クアドラプル | 1111 | 非常に少ない |

これは過去データを分類した結果であり、将来の当選結果を示唆・保証するものではありません。

## ゾロ目（同一数字4桁）についての注意点

「7777」などのゾロ目は人気が高い数字ですが、制度上、次のような特徴があります。

- 購入者が多くなりやすい
- 当選時、当選口数が増えることで1口あたりの当選金額が低くなる場合がある

これは、ナンバーズの当選金が「その回の売上額と当選口数」によって分配される仕組みのためです。

## ナンバーズ4と本アプリの関係について

本サイトで紹介しているアプリは、ナンバーズ4の **過去当選データや購入記録を整理・可視化するための分析・管理ツール** です。
ナンバーズ4は偶然性に基づく宝くじであり、本アプリは当選や利益を保証するものではありません。
あくまで、数字選びや振り返りの参考情報を提供する補助的な位置づけとして設計されています。

### 過去当選データの確認と整理

アプリでは、最新および過去の当選番号を確認できます。
当選履歴を一覧で整理することで、数字の並びや出現パターンを見やすく振り返るための情報整理ツールとして利用できます。

### 機械学習（LSTM）を用いた参考情報の提示

本アプリでは、過去の当選データを入力として機械学習モデル（LSTM）による分析結果を「数字選びの参考となる候補」として表示する機能があります。

- 候補は複数提示され、1つの結果に断定することはありません
- 表示される指標は、あくまで参考目安です
- 抽選結果の当選や収益を保証するものではありません

これらの分析結果は、数字選択を検討する際の一つの視点として活用することを想定しています。

### 購入記録・成績の振り返り（管理機能）

アプリには、ナンバーズ4の購入履歴を記録・管理する機能があります。

- 購入した数字や買い方（ストレート／ボックス／セット等）の記録
- 当選結果や当選金額を反映した収支の整理
- 月次・年次単位での集計や振り返り

これらは購入履歴を後から見返しやすくするための管理機能であり、投資助言や収益保証を目的としたものではありません。

### ゲーミフィケーション要素について

本アプリでは、継続的に利用しやすくするためにレベルやミッションといったゲーム的な要素を取り入れています。
