# ナンバーズ4分析アプリ｜ナンバーズ4予想AI 忍者

**ナンバーズ4予想AI 忍者**は、過去の当選データをAI（LSTM）で分析し、数字選びをサポートする**Android専用の無料分析アプリ**です。統計データに基づいた予測機能と、購入履歴を管理する投資管理機能をオフライン環境でも利用できます。

機械学習（AI）による分析結果を表示する場合がありますが、抽選結果の当選を保証したり、確実な購入方法を示したりするものではありません。

このサイトは、**説明・信頼構築・導線**のための公開ページです。予測ロジックの詳細・学習手法・パラメータ等の内部情報は公開しません。

## 主要機能（概要）

- **当選番号の確認（最新＋過去）**：直近の当選番号や履歴をアプリ内で確認
- **LSTM予測（術）**：過去データをもとに、数字選びの参考となる候補を表示（当選保証はしません）
- **忍者育成（ゲーミフィケーション）**：レベル/ミッション等で術が段階的に解放され、継続しやすい設計
- **購入履歴の記録・管理**：ストレート/ボックス/セット等の購入記録を残し、あとから振り返り可能
- **投資管理（収支・ROIの見える化）**：購入額・当選額・回数などを集計し、月次/年次の成績を確認
- **端末内保存（オフライン中心）**：予測/購入/設定/当選データを端末内に保存し、ネットワーク不調時も継続利用しやすい設計

- 技術について詳しく知りたい方へ  
  → [SQLiteとは？オフライン設計の理由](tech/sqlite.md)

※一部機能はバージョンや設定により提供状況が異なる場合があります。

### こんな方におすすめ

- **データ派のあなたへ**：過去データをExcel等で管理・更新するのが大変な方に。アプリが自動で最新データを統計化します。
- **継続したいあなたへ**：毎回の購入を記録し、収支管理を行うことで、無理のない範囲で宝くじを楽しむ習慣をサポートします。

## LSTM予測（術）について（少し詳しく）

本アプリの予測は、過去当選データを入力として、機械学習モデル（LSTM）で分析した結果を「術」として表示する設計です。

- **候補を複数提示**：1つに断定せず、複数の候補（例：上位候補）を参考情報として提示
- **目安の表示**：候補ごとの“目安”として、確信度や参考指標を表示する場合があります
- **術ごとの違い**：選ぶ術（レベル）によって、利用する情報量や見せ方が変わる設計です
- **オフライン中心**：端末内のデータを使い、ネットワーク不調時でも使えることを重視しています

重要：ここでの「予測」は、抽選結果の当選や利益を保証するものではなく、数字選びの参考情報です。

## 投資管理について（少し詳しく）

もう一つの特徴として、購入記録を「残す → 振り返る」ための投資管理機能があります。

- **購入の記録**：いつ・どの番号を・どの種類（ストレート/ボックス/セット等）で・何口買ったかを記録
- **当選結果の追跡**：当選/非当選や当選金額を反映し、収支の振り返りをしやすくします
- **成績の可視化**：購入額、当選額、回数などを集計し、月次/年次で確認できる設計です

※投資管理は「家計簿」に近い機能であり、当選や収益を保証するものではありません。

## できないこと（誤認防止）

- 当選の保証
- 収益や成果の保証
- 購入の強い推奨（賭博行為の提供はしません）
